# 基本操作

```sh:状態を表示する
git status
```

```sh:ファイルを追加する
git add sample.txt
```

```sh:コミットする
git commit -m "<コメント>#issue番号"
```

```sh:直前のコミットを修正する
git commit --amend
```

```sh:ログを表示する
git log
```

```sh:ログを詳細表示する
git show <commit>
```

```sh:差分を表示する
git diff
```

```sh:クローンする
git clone git@github.com:mabo-102/self-learning_git.git
```

```sh:プッシュする
git push
```

```sh:フェッチする
git fetch
```

```sh:プルする(fetch + merge)
git pull
```
