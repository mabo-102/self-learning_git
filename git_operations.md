# 基本操作

### 状態を表示する

```sh:
git status
```

### ファイルを追加する

```sh:
git add sample.txt
```

### コミットする

```sh:
git commit -m "<コメント>#issue番号"
```

### 直前のコミットを修正する

```sh:
git commit --amend
```

### ログを表示する

```sh:
git log
```

### ログを詳細表示する

```sh:
git show <commit>
```

### 差分を表示する

```sh:
git diff
```

### クローンする

```sh:
git clone git@github.com:mabo-102/self-learning_git.git
```

### プッシュする

```sh:
git push
```

### フェッチする

```sh:
git fetch
```

### プルする(fetch + merge)

```sh:
git pull
```
